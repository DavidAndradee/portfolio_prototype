Ferramentas e Tecnologias

Algumas das skill de desenvolvimento que domíno e utilizo para entender e criar soluções criativas, poderosas e com eficácia

  • HTML 5

    Possibilita desenvolver layouts adaptáveis e de fácil acesso utilizando HTML semântico para organizar o conteúdo da página.

  • CSS 3

    Possibilita aplicar estilos aos elementos HTML por meio do CSS, especificando cores, tipos de letra, espaçamentos e disposição para gerar uma apresentação visual atrativa e uniforme

  • Java Script

    Nescessário para incorporar interatividade às páginas da web, controlar o DOM, responder a eventos e efetuar verificações em formulários.

  • Git

    Experiência com sistemas de controle de versão, como Git, e familiaridade com conceitos como commits, branches e pull requests.

  • Node js

    Experiencia com um ambiente de tempo de execução de código aberto que permite executar JavaScript no servidor, permitindo a criação de aplicativos web escaláveis e de alto desempenho.
